# ミュート機能

不快に感じていたり距離を置きたいと思っていたりするプレイヤーがいる場合は、ミュート機能を利用することでそのプレイヤーのチャットなどを非表示にすることができます。  
生活サーバー内で/muteを実行するか、よもぎ端末から「ミュート」を選択することで機能を利用できます。  

# ミュートの効果

あるプレイヤーに対してミュートを行った際の挙動は以下の通りです。

 - ミュートしたプレイヤーの全体チャットが非表示になります。ただし、自身のチャットは相手に届きます
 - 近距離チャットは互いに届かなくなります
 - ミュートしたプレイヤーの入退出メッセージが表示されなくなります
 - その他、ミュートしたプレイヤーに関する一部システムメッセージが非表示になります
 - 互いに/payでの送金ができなくなります
 - 互いに/mailでのメール送受信ができなくなります
 - 互いに土地保護・土地共有ができなくなります

# ミュートしたら相手にばれる？

ミュートを行った場合でも直ちに相手方に通知されることはありません。  
ただし、送金やメール、土地保護、土地共有を利用した場合は専用のエラーメッセージが表示され実行に失敗するため、ミュートしていることが相手に伝わります。  
また、これらの機能を利用しない場合でも、相手方からの問いかけに返信がないことでミュートを疑われる可能性があります。

# 他プレイヤーにミュートされてしまったら？

ミュートされてしまった場合は、自分自身の言動に問題がなかったかどうかを振り返り、さらに他のプレイヤーからも同様にミュートを受けることの無いよう改善に努めることが大切です。  
インターネット上でのコミュニケーションは相手の顔が見えませんので、外見で差別を受けることはありません。  
運営として特定の言動を禁じるつもりはありませんが、一般論として以下のような言動をとっていないか確認してみてください。  
 - 他のプレイヤーのプライベートに過度に踏み入っている
 - 日頃から強い口調で話している
 - 一方的に自分の話を続ける
 - 第三者を下げていたり、不謹慎であったり差別的であったりする発言を繰り返している
 - 相手方が避けているのにもかかわらずしつこく話しかけている
 - 相手方の言動に対して揚げ足取りをおこなったり、過剰な反応を行う
 - 初対面の人に対してため口を使用している
  
自身がミュートされてしまった場合は、相手方の気持ちを尊重し、距離を置きましょう。  
ミュートを解除してもらうために他の媒体を使用して意思疎通を図ることは、相手が熟考の末ミュートボタンを押したことを鑑みれば、たいていの場合に無意味であるか、なおさら嫌われる原因になります。  
また、運営にミュートの解除を依頼された場合でも、原則として相手方への取次ぎは行いませんのでご了承ください。  
ただし、約束や契約を行ったにもかかわらず相手方が(その約束や契約から逃げるために)ミュートを使用したと考えられる場合は運営までご相談ください。  
運営内で判断を行った後、相手方に約束や契約を履行するよう注意する場合があります。  
  
画面越しであっても相手方は人間です。相手方の気持ちを尊重し、相手方が不快に感じるような言動は避けていただきますようお願いします。

# ミュートに関する例外

 - すでに成立した約束から逃げるためにこの機能を使用することはおやめください。約束(契約)の債務不履行が疑われる場合、運営からご連絡することがあります。
 - 全ての運営をミュートした場合、運営からの連絡が届きませんので利用規約5.5により不利益を受ける可能性があります。

# Discordのミュート

本ページで紹介しているミュート機能は生活サーバー内でのみ有効です。  
Discord上でも距離を置きたい場合は、ユーザーブロック機能を利用したりVCの個別ユーザー音量を0にしたりするなどでご対応ください。
